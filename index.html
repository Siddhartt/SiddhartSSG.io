About Me


My name is Siddhart, I am currently 16 years old and I live in the Netherlands. I started coding when I was 13, I started with visual coding. I didn't like that so I jumped to Unity 3D and c#. In the beginning it was really hard and I lost interest really quick.

A few months later game development got interesting for me again. I gave it a second chance. I found out about Itch.io I found someone who was hosting game jams and I joined their Discord server and made some friends. Someone that I had met was also from the Netherlands and we continued to make games together. Later we started a little game studio called Arctic Studio. Working together on games was a really great and fun experience. I learned a lot about coding and I improved my coding skills.

Nowdays I still try to code every day and improve my coding skills. I worked with C#, Javascript, Python, NodeJS, HTML and a few more languages. I am now trying to get into full stack web development, I also want to do something with that later. This website is the first website I have ever made :D
